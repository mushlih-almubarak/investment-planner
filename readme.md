# 💎 Investment Planner

Craft your premium investment portfolio with elegance.  
Define your salary, allocate across diverse assets, and visualize your financial future.

🔗 **Live Demo:** [https://investment-planner-portfolio.vercel.app](https://investment-planner-portfolio.vercel.app/)  
📂 **Source Code:** [GitHub Repo](https://github.com/mushlih-almubarak/investment-planner)

---

## 🚀 Features

- **Salary Input**  
  Enter your monthly net salary and choose your preferred currency.

- **Investment Allocation**  
  Distribute your investment across multiple asset classes:
  - 📈 Stocks  
  - 🥇 Gold  
  - 📊 ETFs  
  - 🏢 REITs  
  - 💵 Government Bonds  
  - 🏦 Time Deposits  
  - ₿ Cryptocurrency  
  - 🛢 Commodities  

- **Investment Breakdown**  
  Get a clear breakdown of your allocations and visualize your monthly investment.

- **Modern UI/UX**  
  Sleek dark theme with elegant animations.

## 🛠️ Tech Stack

- **Frontend:** React + JSX  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  

## 📦 Installation & Setup

Clone the repository:
```
git clone https://github.com/mushlih-almubarak/investment-planner.git
cd investment-planner
```
Install dependencies:
```
npm install
```
Run the development server:
```
npm run dev
```
Open http://localhost:5173 in your browser.
